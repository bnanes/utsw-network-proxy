You can use the UTSW network proxy (https://myutsouthwestern.swmed.edu) to access subscription-only journal articles from any computer.

  1. Bookmark this link (don't click on it yet):
    UTSW Proxy Link

  2. Navigate to the article you want to read.

  3. Select the bookmark from your bookmarks menu or toolbar.

  4. Log in with your UTSW network credentials.

  5. Read the article.

This should work with any browser on a computer or mobile device. If not, let me know.
